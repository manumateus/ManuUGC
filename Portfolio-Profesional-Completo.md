## 📐 GUÍA VISUAL Y DISEÑO

### **PALETA DE COLORES**
```
🎨 PRIMARY (Confianza + Profesionalidad):
   - Negro profundo: #1a1a1a (textos principales)
   - Blanco: #ffffff (fondo limpio)
   
🎨 ACCENT (Diferenciación):
   - Dorado/Beige: #d4a574 (acentos, hover, líneas divisorias)
   - Gris elegante: #f5f5f5 (fondos secundarios)
   
🎨 PSYCHOLOGY:
   - Transmite: lujo, confianza, profesionalismo, accesibilidad
   - Sin frivolidad, pero cálida
```

### **TIPOGRAFÍA**
```
📝 TÍTULOS (H1, H2):
   - Font: Playfair Display o similar serif elegante
   - Tamaño: 48px (H1), 36px (H2)
   - Peso: 600-700 (Bold)
   - Espaciado: generoso

📝 SUBTÍTULOS / TAGLINE:
   - Font: Inter o Montserrat (sans-serif moderno)
   - Tamaño: 18-20px
   - Peso: 300-400 (Light/Regular)
   - Color: Gris oscuro con toque elegancia

📝 BODY TEXT:
   - Font: Inter, Open Sans o similar
   - Tamaño: 16px
   - Peso: 400 (Regular)
   - Alto de línea: 1.6 (legibilidad)
   - Color: #1a1a1a

📝 CTA BUTTONS:
   - Font: Montserrat o Inter
   - Tamaño: 14-16px
   - Peso: 600 (Semibold)
   - Transformación: mayúsculas, letra tracking
```

### **ELEMENTOS DE DISEÑO**
```
✨ ESPACIADO:
   - Margen superior/inferior de secciones: 80-120px
   - Padding interno: 40-60px
   - Espaciado entre líneas de texto: 1.6-1.8

✨ LÍNEAS DIVISORIAS:
   - Color: Dorado/Beige (#d4a574)
   - Altura: 2px
   - Uso: Separar secciones de forma elegante (no abuso)

✨ CARDS / CONTENEDORES:
   - Border radius: 4px (muy sutil, casi recto)
   - Sombra: Muy suave (0 4px 12px rgba(0,0,0,0.08))
   - Background: Blanco o gris muy claro (#f9f9f9)
   - Padding: 32px

✨ TRANSICIONES:
   - Duración: 0.3s - 0.5s
   - Timing: ease-in-out
   - Efectos: Hover sutiles (cambio color accent, elevación mínima)

✨ IMÁGENES:
   - Ratio: Variado según contenido
   - Filtro: Levemente desaturado o con overlay sutil para consistencia
   - Bordes: Sin bordes, integradas naturalmente
```

# 📝 COPY GUIDELINES Y TONO

## **REGLAS DE ORO**

1. **Cercano + Experto**
   - Usa "yo" y "mi" (no somos empresa)
   - Pero muestra credibilidad con datos/skills
   - Evita clichés: No digas "apasionada por crear contenido"
   - Sí di: "Creo contenido que vende porque entiendo por qué la gente compra"

2. **Enfocado en Valor, no Features**
   - No: "Creo videos profesionales"
   - Sí: "Creo videos que generan conversión porque están diseñados psicológicamente para ello"

3. **Concisión**
   - Párrafos máximo 3-4 líneas
   - Frases impactantes, no párrafos largos
   - Pull quotes para ideas clave

4. **Realidad, no Promesas**
   - Evita: "Garantizo que tus ventas subirán"
   - Sí: "Mi contenido está diseñado para conversión"

5. **CTA Claro**
   - Siempre botones visibles
   - Microcopy que explique qué pasa después
   - Múltiples opciones (email, brief, whatsapp)
