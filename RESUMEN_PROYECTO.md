## 🚀 PRÓXIMOS PASOS (SÚPER FÁCIL)

### PASO 1: Preparar tu Contenido (15 min)

Crea carpetas en `public/`:

```
public/
├── images/
│   ├── hero-image.jpg           ← Tu foto profesional
│   ├── about-photo.jpg          ← Tu foto para sección about
│   └── portfolio/
│       ├── portfolio-1.jpg      ← Portada video 1
│       ├── portfolio-2.jpg
│       └── ... (6-10 imágenes)
├── videos/
│   └── portfolio/
│       ├── portfolio-1.mp4      ← Tu video UGC 1
│       ├── portfolio-2.mp4
│       └── ... (tus videos)
└── logos/
    └── (logos de marcas)
```

**Tamaños recomendados:**
- Imágenes: 1200x800px (optimizadas < 200KB)
- Videos: 1080x1920 (vertical) o 1920x1080 (horizontal), MP4

### PASO 2: Actualizar JSON Data (10 min)

Edita `src/data/portfolio.json`:

```json
[
  {
    "id": "project-1",
    "title": "Charlotte Tilbury - Beauty Tutorial",
    "category": "product",
    "description": "Product demo beauty",
    "image": "/images/portfolio/portfolio-1.jpg",
    "video": "/videos/portfolio/portfolio-1.mp4",
    "brands": ["Charlotte Tilbury"],
    "tags": ["Belleza", "Tutorial"]
  },
  // ... repite para cada proyecto
]
```

### PASO 3: Instalar y Probar (5 min)

```bash
cd c:\Escritorio\ManuUGC\ManuUGC

# Instalar dependencias
npm install

# Prueba local
npm run dev

# Abre http://localhost:3000
```

### PASO 4: Elegir Hosting y Deploy (10 min)

#### OPCIÓN A: Netlify (Recomendado)
1. Sube a GitHub
2. Conecta repo en netlify.com
3. Build: `npm run build`
4. Publish: `dist`
5. Deploy automático ✅

#### OPCIÓN B: Vercel
1. Sube a GitHub
2. Import en vercel.com
3. Auto-detecta Astro
4. Deploy ✅

#### OPCIÓN C: Hosting Tradicional
1. `npm run build`
2. Descarga `dist/`
3. Sube via FTP
4. Deploy ✅

### PASO 5: Configurar Dominio (5 min)

- Compra dominio (ej: manuelamateus.com)
- Conecta en Netlify/Vercel o tu hosting
- SSL automático ✅

### PASO 6: Integrar Formularios (5 min)

#### Opción: Formspree
1. Ve a formspree.io
2. Crea account
3. Genera form ID
4. Copia en `CTA.astro`
5. Los emails llegan a tu inbox ✅

## 💡 TIPS PARA MAXIMIZAR CONVERSIÓN

### En Portfolio
- Muestra 6-12 videos mínimo
- Ordena por tipo (product, lifestyle, ads, etc.)
- Incluye logos de marcas conocidas
- Videos deben ser de calidad profesional

### En CTAs
- "Enviar Brief" es mejor que "Contactar"
- Responde en 24h (como prometes)
- Haz follow-up después de 3 días si no contesta

### En Copy
- Evita clichés ("apasionada por crear contenido")
- Enfatiza resultados ("contenido que convierte")
- Muestra confianza sin arrogancia
- Usa números si tienes (3 años exp, 50+ proyectos, etc.)

### En SEO
- Google Analytics para tracking
- Focus keywords: "creadora UGC", "content creator", "UGC videos"
- Semanal: revisa keywords ranking
- Mensual: actualiza portfolio con nuevos proyectos
